Original system / Market-failure logic

Every chart finds a bottom.

Bodenza uses its own code and drawing rules. Each seed chooses from six market-failure systems: collision-aware crash flows, capitulation clouds, chart scribbles, fractured order books, topographic selloffs, and distressed tape fields. Under every abstraction is a generated price path that bleeds, gaps, bounces, or capitulates.

The system uses the Bobo palette only: #FFFFFF, #472C1B, #6F452D, #B90B2A, and #000000. It studies established generative techniques while remaining an original Bobo project that is not affiliated with Tyler Hobbs, Fidenza, Art Blocks, or the artists behind any reference work.
